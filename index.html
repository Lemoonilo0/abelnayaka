DNS dan DNS Server

DNS (Domain Name System) adalah sistem yang menerjemahkan nama domain menjadi alamat IP. Sistem ini bekerja seperti "buku telepon" internet, memudahkan kita mengakses website tanpa perlu mengingat deretan angka IP.

Fungsi Utama DNS:

  • Konversi nama domain ke alamat IP
  • Penyimpanan cache untuk akses lebih cepat
  • Load balancing untuk distribusi beban server
  • Redundansi untuk ketersediaan tinggi

Komponen DNS Server:

  • Recursive Resolver: Server yang menangani permintaan dari client
  • Root Server: Pusat hierarki sistem DNS global
  • TLD Server: Mengelola domain tingkat atas (.com, .net, dll)
  • Authoritative Server: Menyimpan record DNS spesifik
